Finds the best route from a source node to a destination node, considering both driving and walking.

This function calculates the optimal route from a source node to a destination node by first driving to a parking node and then walking to the destination. It considers a maximum walking time and avoids specified nodes and segments.

Parameters
sourceThe ID of the source node.
destinationThe ID of the destination node.
max_walking_timeThe maximum allowed walking time.
avoid_nodesA set of node IDs to avoid in the route.
avoid_segmentsA vector of node ID pairs representing segments to avoid in the route.
